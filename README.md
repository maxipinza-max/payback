# Payback Calculator

Calculadora para proyectar el margen mensual de un producto y estimar en que mes se recupera la inversion de desarrollo.

## Funcionalidades

- Calcula margen de contribucion y margen mensual.
- Proyecta ingresos, costos y margen acumulado por mes.
- Identifica el mes exacto de payback.
- Muestra ROI acumulado al cierre del horizonte elegido.
- Incluye una vista grafica y una tabla mensual de detalle.
- Mantiene la seccion existente de estimaciones de proyectos.

## Tecnologia

- Angular 21
- Nx monorepo
- NestJS para servicios auxiliares del proyecto base

## Uso local

```bash
npm install
npm run start:web
```

La app queda disponible por defecto en `http://localhost:4200`.
